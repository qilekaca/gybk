## 介绍

**职务名称:** **生鲜**（蔬菜、水果、水产、肉、主食、烘焙、熟食、散干货蛋品）、**食品**（散零食、日配、酒饮、粮油）、**非食**（针服、清洁、洗化、电商、收银服务、杂摆、加百）<br>

**直接上级:** 本部门处长<br>

**直接下属:** 理货员、促销员、力工<br>

**本职工作:** 本课的第一责任人，全权负责本课的商品管理、员工管理，促成各项管理及营运指标的达成。<br>
